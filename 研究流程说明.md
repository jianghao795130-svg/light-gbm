# 研究流程说明

1. 读取 all_factors_kline.pkl，并执行预过滤：剔除 ST、上市不足 250 天、次日无法买入样本。
2. 读取 factor_未来n日涨跌_5.pkl 作为原始收益标签，其他 factor_*.pkl 作为因子。
3. 对原始收益和数值因子按交易日做截面排序，同时保留 info_cols 和 一级行业。
4. 按交易日对未来收益和数值因子先做 MAD(5) 去极值，再做市值 + 行业中性化，最后做普通标准化（减去截面均值并除以截面标准差），行业哑变量使用 drop_first=True。
5. 训练标签使用 未来收益_预处理，原始收益保存在 raw_return 中用于 IC 评估。
6. 训练特征只使用排序后的数值因子列，一级行业与 info_cols 不进入训练。
7. 使用 6 年训练、12 个月验证、3 个月测试、3 个月步长的滚动窗口。
8. 每个窗口训练 5 个不同随机种子的 LightGBM 回归模型，对测试集预测取平均作为 ml_factor。
9. 用 RMSE 曲线和 Valid/Train RMSE 比例检查过拟合。
10. 使用 RankIC、Pearson IC、相对全市场等权的 RankIC、TopN 收益曲线、月度收益表评估因子。
11. 输出 split / gain / SHAP 综合重要性，并识别低重要性噪声因子。
12. 研究输出文件统一保存在 research_outputs 目录。
